A clean and responsive weather app that gives you real-time weather data and a 3-day forecast for any city in the world.

---

## Features

- 🔍 Search weather by city name
- 🌡️ Current temperature, condition, humidity, wind speed & precipitation
- 📅 3-day weather forecast
- ❄️ Animated loading screen
- 📱 Fully responsive — works on mobile, tablet, and desktop

## Tech Stack

- **HTML, CSS, JavaScript** — no frameworks, pure vanilla
- **WeatherAPI** — for real-time weather and forecast data
- **Bootstrap Icons** — for UI icons

## Getting Started

1. Clone the repo

   ```bash
   git clone https://github.com/your-username/weatherpin.git
   ```

2. Get a free API key from [weatherapi.com](https://www.weatherapi.com/)

3. Open `script.js` and replace the API key:

   ```js
   let API = `your_api_key_here`;
   ```

4. Open `index.html` in your browser — done!
